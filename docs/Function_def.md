# 역할별 기능 명세서

## 박건일 PGI

### 커뮤니티 기능 (댓글 제외)

- 게시판
    - 필터링 탭으로 [전체/자유게시판/팀 구하기/건의하기] 카테고리로 나뉜다.
    - 게시글 정렬: 조회수, 최신 순
    - 게시글 카드 내부에 제목, 내용, 닉네임, 생성일자, 카테고리, 조회수 표시
    - 각 게시글 별로 사용자들 댓글 가능
    - 작성자는 게시글을 등록, 수정, 삭제 가능
        - 건의하기는 사용자가 등록 시 모든 사용자가 볼 수 있고 관리자에 의해 비밀글 처리 될 수 있다.
- 게시글 필터링 방식
    - 미리 정의된 욕설 사전에서 텍스트 단순 비교
- 사용자 신고 기능
    - 사용자는 서비스, 피드백 , 커뮤니티, 커뮤니티 댓글을 신고할 수 있다.
    - 신고를 누르면 모달창을 띄어 신고 사유를 선택하거나 입력할 수 있다.
- 신고 모달 창에 뜨는 신고 유형. 아래 5가지 유형 중 하나 선택
    - 불법/음란물 광고: 법률 위반 광고 및 불법 링크 포함된 경우
    - 도배 및 스팸: 홍보 시스템 악용 및 지속적인 도배한 경우
    - 욕설 및 비방: 타인에 대한 인신공격, 모욕, 혐오 표현한 경우
    - 표절 및 도용: 타인의 소스코드, 아이디어 무단 도용한 경우
    - 기타 및 상세 사유(직접 입력)
- 관리자 페이지
    - 신고 내역 확인
        - 신고한 순으로 전체 신고내역을 띄운다.
        - 전체, 서비스, 피드백, 커뮤니티, 커뮤니티 댓글 탭으로 나뉜다.
            - 각 신고 내역 별로 신고유형, 신고사유, 처리상태(승인됨, 거부됨) 확인 가능
        - 
    - 정책을 위반하면 해당 게시물 블라인드 (버튼)
        - 신고 승인(APPROVED) 처리 시 target_id의 게시글을 블라인드 처리한다.

## 박재범 PJB

### 빡낌(포인트) 기능
    - 현재 로그인한 사용자의 빡낌 잔액을 헤더에 전역 표시한다.
- 빡낌 변동 내역을 최신순 페이지네이션으로 조회할 수 있다. (마이페이지에서)
    
    **위치:** 마이페이지 → 내 빡낌 → 자세히 보기
    
- 출석체크
    - 마이페이지에서 ‘출석’ 버튼 누르면 출석 (마이페이지 좌측 하단)
    - 출석 체크 시 하루 1회 빡낌이 지급된다. (KST 기준)
    - 연속 출석 7 / 15 / 30일 달성 시 보너스 빡낌 +200이 추가 지급된다.
    - **버튼 아래 (항상 보임)**: 연속 N일 출석 중
    - **토스트 (출석 직후)**: 오늘 받은 빡낌 내역 + 현재 잔액
    - 하루라도 빠지면 연속 출석이 초기화
    - 하루 1회만 출석 가능하며 이미 출석한 경우 버튼이 비활성화된다.
- 홍보
    - 홍보 상품 목록을 조회하고 빡낌으로 홍보를 신청할 수 있다.
    - 홍보 슬롯은 선착순이며, 슬롯 소진 시 신청이 불가하다.
    - 빡낌 획득/차감 시 헤더 잔액이 자동으로 갱신된다.
    - 홍보 상품 전체 목록과 단건 상세를 조회할 수 있다.
    - 홍보 신청 시 빡낌이 즉시 차감되고 홍보가 활성화된다.
    - 카드 상태는 2종  1.신청 가능: 노란색 2.빡낌 부족 : 빨간색, 슬롯 부족: 빨간색
    - 카드 내 기간 선택  (1일 / 3일 / 7일 - 1차 배포는 1일차만 활성)
    - 신청하기 클릭하면 → 모달 창 띄움(2단으로)
    - 좌측 — 신청 폼:
        - 상품명 + 가격
        - 홍보할 서비스 드롭다운
        - 이미 홍보 중인 서비스 → "홍보중" 라벨 + 선택 불가
        - 빡낌 요약 (서비스 선택 후 표시):
        - 현재 빡낌 / 차감 빡낌(빨간색) / 차감 후 잔액
        - [취소] [신청 확인]
    - 우측 — 미리보기:
    - 서비스 미선택: "서비스를 선택하면 노출 모습을 확인할 수 있어요"
    - 서비스 선택 후: 해당 상품 노출 위치 축소 미리보기
- 빡낌 정책
    - 빡낌 잔액이 0 이하로 내려갈 수 없다.
    - 빡낌 내역은 삭제하거나 수정할 수 없다.
    - 지급된 빡낌은 회수하지 않는다. (피드백 삭제 시에도)
- 홍보 정책
    - 사용자는 본인 서비스만 홍보 신청할 수 있다.
    - 홍보 신청은 즉시 활성화되며 환불되지 않는다.
    - 같은 서비스에 같은 상품을 중복 신청할 수 없다.
    - 같은 서비스를 다른 상품으로 동시에 여러 곳에 홍보할 수 없다.
    - 홍보 중인 서비스가 비활성/차단되면 홍보가 중단된다.
    - 슬롯이 소진된 상품은 신청할 수 없다.
    - 홍보 기간이 만료되면 자동으로 종료됨 처리된다.

### 2. 피드백
- 로그인한 사용자만 피드백 작성 및 등록 가능
- 자신이 게시한 서비스에 피드백 작성 불가
- 가입 24시간 지난 사용자의 첫 피드백 등록→ 서비스 등록 조건 충족
    - 최초 1회 모달 뜸
        - [축하합니다! 서비스를 등록할 수 있어요]
        ☑️ 가입 24시간 경과
        ☑️ 피드백 1건 작성
        [내 서비스 등록하러 가기→]
- 작성 횟수 규제
    - 1일 피드백 3건 작성 제한
    - 서비스 1건당 피드백 1건 작성 제한
- 빡낌 지급 정책
    - 피드백 1건 당 200빡낌 지급

    - 품질 검사: LMM으로 진행
- 삭제 클릭 시 모달 뜸
    - 작성 30일 미만
        - [아직 피드백을 삭제할 수 없습니다.
        작성 30일 이후부터 삭제 가능해요.]

        - [개발자에게 소중한 자산이 될 수 있어요.
        정성껏 작성하신 피드백인데 정말 삭제하시겠어요?
        지급됐던 빡낌은 회수되지 않습니다.]

## 김은경 KEG

### 1. 마이페이지 기능
- 사용자는 자신의 프로필 이미지, 닉네임, 연동 계정 정보를 확인할 수 있다.
- 사용자는 프로필 수정 화면으로 이동할 수 있다.
- 사용자는 현재 보유 빡낌, 총 적립 빡낌, 총 사용 빡낌을 확인할 수 있다.
- 사용자는 출석체크를 진행하고, 출석 여부 및 연속 출석 보너스를 확인할 수 있다.
- 사용자는 내 서비스, 내 피드백, 팔로워, 팔로잉, 내 게시글, 내 댓글 수를 요약 통계로 확인할 수 있다.
- 사용자는 팔로잉 및 팔로워 목록 일부를 확인하고 전체 목록으로 이동할 수 있다.
- 사용자는 자신이 작성한 게시글 목록과 댓글 목록 일부를 확인하고 전체 목록으로 이동할 수 있다.
- 사용자는 자신이 등록한 서비스 목록 일부를 확인하고 전체 목록으로 이동할 수 있다.
- 사용자는 자신이 작성한 피드백 목록 일부를 확인하고 전체 목록으로 이동할 수 있다.

### 2. 커뮤니티 댓글 기능
- 사용자는 각 게시글 별로 댓글을 조회할 수 있다.
- 로그인한 사용자는 게시글에 댓글을 작성할 수 있다.
- 댓글에는 작성자의 닉네임, 댓글 내용, 작성일자가 표시된다.
- 사용자는 본인이 작성한 댓글을 수정할 수 있다.
- 사용자는 본인이 작성한 댓글을 삭제할 수 있다.
- 댓글 작성자 본인에게만 수정, 삭제 버튼이 표시된다.

### 3. 팔로우 / 팔로잉 기능
- 사용자는 다른 사용자를 팔로우할 수 있다.
- 사용자는 이미 팔로우한 사용자를 언팔로우할 수 있다.
- 사용자는 자신이 팔로우한 사용자 목록을 확인할 수 있다.
- 사용자 프로필에는 팔로워 수와 팔로잉 수가 표시된다.
- 이미 팔로우한 사용자는 팔로우 버튼 대신 팔로잉 또는 언팔로우 버튼이 표시된다.
- 사용자는 자기 자신을 팔로우할 수 없다.
- 중복 팔로우는 허용하지 않는다.

## 조현정 CHJ

### 서비스 등록

- 헤더에서 ‘서비스’ 버튼 클릭시 서비스 페이지 진입
- 서비스 페이지
    - 왼쪽에 카테고리 별 메뉴
    - 오른쪽에 전체 서비스 조회
        - 한 페이지에 10개씩 / 2열 5행
        - 서비스마다 썸네일 이미지, 서비스명, 한 줄 설명
- 서비스 카테고리
    - IT · 소프트웨어 · AI
    - 요식업 · 푸드테크 · 카페
    - 제조 · 생산 · 하드웨어
    - 교육 · 에듀테크 · 지식 콘텐츠
    - 금융 · 핀테크 · 법률 · 세무
    - 엔터테인먼트 · 미디어 · 예술
    - 라이프스타일 · 여행 · 숙박
    - 뷰티 · 패션 · 이커머스
    - 마케팅 · 광고 · 디자인
    - 바이오 · 헬스케어 · 환경
- 첫 서비스 등록 조건 확인
    - 로그인+관심 카테고리 설정 이후 첫 서비스 등록 체크리스트 모달 뜸
        - 조건 1: 24시간 이후, 조건 2: 피드백 1개 작성
    - 조건 달성 될 때마다 체크리스트 모달뜸 (조건 체크되어서)
    - 두 조건 다 충족시 ‘조건 충족 완료’ 텍스트. 하단에 ‘서비스 등록하러 가기’ 버튼 생김
- 서비스 등록 페이지 진입
    - 1. 서비스 페이지 카테고리 별 메뉴 위에 ‘내 서비스 등록’ 버튼 클릭
    - 2. 체크리스트 모달에서 ‘서비스 등록하러 가기’ 버튼 클릭
- 서비스 등록 시 사용자 입력
    - 서비스명
    - 서비스 한 줄 소개
    - 서비스 설명
    - 서비스 URL
    - 서비스 카테고리
    - 서비스 썸네일 (선택사항) : 1280×720, 3MB 이하, PNG, JPG
- AI 도우미 기능
    - 토글 버튼으로 켜고 끄기 가능
    - 켜면 사용자가 입력한 서비스에 대한 간단한 설명을 가지고 LLM이 ‘한 줄 소개’와 ‘서비스 설명’, ‘카테고리’ 추천
- 서비스 URL 검증
    - Google Web Risk 툴 사용
