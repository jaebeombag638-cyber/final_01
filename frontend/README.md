## 🛠️ 프론트엔드 시작하기 (Frontend Quick Start)

팀원들은 레포지토리를 클론(Clone)받은 후, 터미널에서 다음 명령어를 순서대로 입력하여 로컬 개발 서버를 구동해 주세요.

```bash
# 1. 프론트엔드 디렉토리로 이동
cd frontend

# 2. 프로젝트 필수 패키지 설치
npm install

# 3. 로컬 개발 서버 실행
npm run dev
구동 후 터미널에 뜨는 주소(
